# Dart Data Types & Variables: A Comprehensive Guide for Beginners

Dart, the language behind Flutter, is a strongly-typed language, meaning that every variable must have a declared type. Understanding data types and variables is fundamental to writing effective Dart code. This guide will walk you through the core concepts, providing clear explanations and practical examples to help you master them.

## What are Data Types?

Data types classify the kind of data a variable can hold. They tell the compiler how to interpret and store the data. Dart offers a rich set of built-in data types, which can be broadly categorized as follows:

- **Numbers:** Used to represent numerical values.
- **Strings:** Used to represent textual data.
- **Booleans:** Used to represent true/false values.
- **Lists:** Used to represent ordered collections of items.
- **Maps:** Used to represent key-value pairs.
- **Runes:** Used to represent Unicode characters.
- **Symbols:** Used to represent identifiers.
- **Null:** Represents the absence of a value.

Let's dive deeper into each of these data types.

## 1. Numbers: `int` and `double`

Dart has two primary data types for numbers:

- **`int`:** Represents integers, which are whole numbers without decimal points.

  ```plaintext
  void main() {
    int age = 30;
    int numberOfStudents = 150;

    print('Age: $age');
    print('Number of Students: $numberOfStudents');
  }
  ```

- **`double`:** Represents double-precision floating-point numbers, which are numbers with decimal points.

  ```plaintext
  void main() {
    double price = 29.99;
    double pi = 3.14159;

    print('Price: $price');
    print('PI: $pi');
  }
  ```

**Important Note:** Dart also has a `num` type, which is the parent class of both `int` and `double`. You can use `num` if you want a variable to be able to hold either integer or floating-point values.

```plaintext
void main() {
  num myNumber = 10; // Initially an integer
  print(myNumber.runtimeType); // Output: int

  myNumber = 3.14;   // Now a double
  print(myNumber.runtimeType); // Output: double
}
```

## 2. Strings: `String`

The `String` data type is used to represent textual data. Strings are sequences of characters enclosed in single quotes (`'`) or double quotes (`"`).

```plaintext
void main() {
  String name = 'John Doe';
  String message = "Hello, Dart!";

  print('Name: $name');
  print('Message: $message');
}
```

**String Interpolation:** Dart allows you to embed variables and expressions directly into strings using string interpolation. Use the `$` symbol followed by the variable name. For more complex expressions, enclose them in `${}`.

```plaintext
void main() {
  String firstName = 'Alice';
  String lastName = 'Smith';
  String fullName = '$firstName $lastName'; // String interpolation

  int age = 25;
  String ageMessage = 'Alice is ${age + 5} years old in 5 years.';

  print('Full Name: $fullName');
  print(ageMessage);
}
```

**Multiline Strings:** You can create multiline strings using triple single quotes (`'''`) or triple double quotes (`"""`).

```plaintext
void main() {
  String longText = '''
    This is a multiline string.
    It can span across multiple lines
    without needing to be escaped.
  ''';

  print(longText);
}
```

## 3. Booleans: `bool`

The `bool` data type represents truth values: `true` or `false`.

```plaintext
void main() {
  bool isLoggedIn = true;
  bool isStudent = false;

  print('Is logged in: $isLoggedIn');
  print('Is student: $isStudent');
}
```

Booleans are commonly used in conditional statements (e.g., `if`, `else`) and loops.

```plaintext
void main() {
  bool isRaining = true;

  if (isRaining) {
    print('Bring an umbrella!');
  } else {
    print('Enjoy the sunshine!');
  }
}
```

## 4. Lists: `List`

The `List` data type represents an ordered collection of items. Lists are also known as arrays in other programming languages.

```plaintext
void main() {
  List<String> fruits = ['apple', 'banana', 'orange'];
  List<int> numbers = [1, 2, 3, 4, 5];

  print('Fruits: $fruits');
  print('Numbers: $numbers');

  // Accessing elements by index:
  print('First fruit: ${fruits[0]}'); // Output: First fruit: apple
}
```

**List Operations:** Dart provides various methods for manipulating lists, such as:

- `add()`: Adds an element to the end of the list.
- `insert()`: Inserts an element at a specific index.
- `remove()`: Removes the first occurrence of an element.
- `removeAt()`: Removes the element at a specific index.
- `length`: Returns the number of elements in the list.

```plaintext
void main() {
  List<String> colors = ['red', 'green', 'blue'];
  colors.add('yellow');           // Add 'yellow' to the end
  colors.insert(1, 'purple');    // Insert 'purple' at index 1
  colors.remove('red');           // Remove 'red'
  colors.removeAt(2);         // Remove the element at index 2

  print('Colors: $colors'); // Output: Colors: [green, purple, yellow]
  print('Length: ${colors.length}'); // Output: Length: 3
}
```

## 5. Maps: `Map`

The `Map` data type represents a collection of key-value pairs. Each key in a map must be unique.

```plaintext
void main() {
  Map<String, String> person = {
    'name': 'Alice',
    'age': '30',
    'city': 'New York'
  };

  print('Person: $person');

  // Accessing values by key:
  print('Name: ${person['name']}'); // Output: Name: Alice
}
```

**Map Operations:** Dart provides methods for working with maps, including:

- `[]`: Accesses the value associated with a key.
- `containsKey()`: Checks if a map contains a specific key.
- `containsValue()`: Checks if a map contains a specific value.
- `remove()`: Removes a key-value pair from the map.
- `keys`: Returns an iterable of the map's keys.
- `values`: Returns an iterable of the map's values.

```plaintext
void main() {
  Map<String, int> scores = {
    'Alice': 90,
    'Bob': 85,
    'Charlie': 95
  };

  print('Alice\'s score: ${scores['Alice']}');
  print('Contains Bob: ${scores.containsKey('Bob')}');
  print('Contains score 90: ${scores.containsValue(90)}');

  scores.remove('Bob');

  print('Keys: ${scores.keys}');
  print('Values: ${scores.values}');

  print('Updated Scores: $scores');
}
```

## 6. Runes: `Runes`

Runes represent Unicode code points. They're used for representing characters that can't be represented by a single 16-bit code unit (especially useful for handling emoji and complex characters).

```plaintext
void main() {
  String heartEmoji = '\u2665'; // Unicode heart symbol
  print(heartEmoji);

  Runes input = new Runes('\u2665  \u{1f600}  \u{1f47b}'); //Heart, grinning face, ghost

  print(String.fromCharCodes(input));
}
```

## 7. Symbols: `Symbol`

Symbols are opaque identifiers used to represent the names of variables or methods. They are less commonly used in everyday Dart programming, but can be useful in metaprogramming.

```plaintext
void main() {
  Symbol mySymbol = #myVariable;

  print(mySymbol); // Output: Symbol("myVariable")
}
```

## 8. Null: `Null`

The `Null` type has only one value: `null`. It represents the absence of a value. Dart has strong null safety which requires you to explicitly declare variables that can hold `null` or use features like the null-aware operator (`?`) to prevent unexpected errors.

```plaintext
void main() {
  String? nullableName; // Can hold a String or null

  print(nullableName); // Output: null

  String name = nullableName ?? 'Guest'; // Using null-aware operator

  print(name); //Output: Guest

}
```

## Declaring Variables

In Dart, you can declare variables using the following keywords:

- **`var`:** The `var` keyword allows you to declare a variable without explicitly specifying its data type. Dart infers the type based on the initial value assigned to the variable.

  ```plaintext
  void main() {
    var name = 'Alice'; // Dart infers the type to be String
    var age = 30;       // Dart infers the type to be int
    var price = 9.99;    // Dart infers the type to be double

    print('Name: $name, Type: ${name.runtimeType}');
    print('Age: $age, Type: ${age.runtimeType}');
    print('Price: $price, Type: ${price.runtimeType}');
  }
  ```

  Once the type is inferred by `var`, it cannot be changed to a different type.

- **Explicit Typing:** You can explicitly declare the data type of a variable. This is often preferred for code clarity and maintainability.

  ```plaintext
  void main() {
    String name = 'Bob';
    int age = 25;
    double height = 1.75;
    bool isStudent = true;

    print('Name: $name');
    print('Age: $age');
    print('Height: $height');
    print('Is Student: $isStudent');
  }
  ```

- **`dynamic`:** The `dynamic` keyword allows a variable to hold values of any type. However, using `dynamic` reduces type safety and can lead to runtime errors if you're not careful.

  ```plaintext
  void main() {
    dynamic myVariable = 10;
    print(myVariable.runtimeType); // Output: int

    myVariable = 'Hello';
    print(myVariable.runtimeType); // Output: String
  }
  ```

- **`final` and `const`:** These keywords are used to declare variables that cannot be reassigned after they are initialized.

  - `final`: The value of a `final` variable must be known at runtime.

    ```plaintext
    void main() {
      final String username = 'johndoe';
      // username = 'newname';  // Error: Cannot assign to final variable 'username'.
      print(username);
    }
    ```

  - `const`: The value of a `const` variable must be known at compile time. `const` variables are implicitly `final`. They are typically used for constants that are known at compile time, such as mathematical constants or configuration values.

    ```plaintext
    void main() {
      const double PI = 3.14159;
      // PI = 3.14;  // Error: Constant variables can't be assigned a value.

      print('PI: $PI');
    }
    ```

    `const` can also be used to create compile-time constant values of lists or maps.

    ```plaintext
    void main() {
        const myList = [1, 2, 3];
        //myList[0] = 4; // Error: Unsupported operation: Cannot modify an unmodifiable list

        const myMap = {'a': 1, 'b': 2};
       // myMap['a'] = 3; //Error: Unsupported operation: Cannot modify an unmodifiable map
    }
    ```
