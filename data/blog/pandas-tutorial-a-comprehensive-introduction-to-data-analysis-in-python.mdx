# Pandas Tutorial: A Comprehensive Introduction to Data Analysis in Python

Pandas is a powerful and versatile Python library that provides data structures and data analysis tools that make working with structured (tabular, relational) and unstructured data easy and intuitive.  It's the cornerstone of many data science workflows and a must-know for anyone working with data in Python. This tutorial will walk you through the basics of Pandas, covering DataFrames, Series, data manipulation, and how to get started.

## What is Pandas?

Pandas is built on top of NumPy, another fundamental Python library for numerical computation.  It introduces two main data structures:

*   **Series:** A one-dimensional labeled array capable of holding any data type (integers, strings, floating point numbers, Python objects, etc.). Think of it like a column in a spreadsheet.

*   **DataFrame:** A two-dimensional labeled data structure with columns of potentially different types.  It's similar to a spreadsheet or SQL table.  DataFrames are the workhorse of Pandas, providing a powerful and flexible way to organize and manipulate data.

## Why use Pandas?

Pandas offers several advantages for data analysis:

*   **Data Cleaning & Preparation:**  Pandas excels at handling missing data, filtering data, and transforming data into a usable format.
*   **Data Exploration & Analysis:** You can easily calculate summary statistics, group data, and perform various analytical operations.
*   **Data Visualization:** Pandas integrates well with libraries like Matplotlib and Seaborn, allowing you to create visualizations of your data.
*   **Data Storage & Retrieval:**  Pandas can read and write data from various formats like CSV, Excel, SQL databases, and more.
*   **Speed & Efficiency:**  Pandas is optimized for performance, allowing you to work with large datasets efficiently.

## Understanding Pandas Series

A Pandas Series is a one-dimensional labeled array.  Let's create a simple Series:

```python
import pandas as pd

# Creating a Series from a list
data = [10, 20, 30, 40, 50]
series = pd.Series(data)
print(series)
```

This will output:

```
0    10
1    20
2    30
3    40
4    50
dtype: int64
```

Notice that Pandas automatically assigns an index to each element in the Series, starting from 0.  We can customize the index:

```python
import pandas as pd

# Creating a Series with a custom index
data = [10, 20, 30, 40, 50]
index = ['a', 'b', 'c', 'd', 'e']
series = pd.Series(data, index=index)
print(series)
```

Output:

```
a    10
b    20
c    30
d    40
e    50
dtype: int64
```

Now, we can access elements using the index:

```python
print(series['b'])  # Output: 20
```

## Introducing Pandas DataFrames

A DataFrame is a two-dimensional labeled data structure with columns of potentially different types.  Think of it as a spreadsheet.

### Creating a DataFrame

Let's create a DataFrame from a dictionary:

```python
import pandas as pd

data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
    'Age': [25, 30, 27, 22],
    'City': ['New York', 'London', 'Paris', 'Tokyo']
}

df = pd.DataFrame(data)
print(df)
```

Output:

```
      Name  Age      City
0    Alice   25  New York
1      Bob   30    London
2  Charlie   27     Paris
3    David   22     Tokyo
```

Each key in the dictionary becomes a column in the DataFrame, and the values associated with each key become the data in that column.

### Accessing Data in a DataFrame

*   **Selecting Columns:**

    ```python
    print(df['Name'])  # Selects the 'Name' column
    ```

    Output:

    ```
    0      Alice
    1        Bob
    2    Charlie
    3      David
    Name: Name, dtype: object
    ```

*   **Selecting Rows (using `loc` and `iloc`):**

    *   `loc`:  Accesses rows based on labels (index values).
    *   `iloc`: Accesses rows based on integer position (0-based indexing).

    ```python
    print(df.loc[0])   # Selects the first row (index 0)
    print(df.iloc[2])  # Selects the third row (index 2)
    ```

    Output:

    ```
    Name       Alice
    Age           25
    City    New York
    Name: 0, dtype: object
    Name     Charlie
    Age           27
    City       Paris
    Name: 2, dtype: object
    ```

*   **Selecting Rows and Columns:**

    ```python
    print(df.loc[0, 'Name'])  # Selects the 'Name' value in the first row
    print(df.iloc[1, 2])   # Selects the City value in the second row (index 1) and third column (index 2)
    ```

    Output:

    ```
    Alice
    London
    ```

### DataFrame Attributes and Methods

Pandas DataFrames have many useful attributes and methods:

*   **`shape`:**  Returns the dimensions of the DataFrame (rows, columns).

    ```python
    print(df.shape) # Output: (4, 3) - 4 rows, 3 columns
    ```

*   **`head()`:**  Returns the first *n* rows (default is 5).

    ```python
    print(df.head(2)) # Output: First 2 rows
    ```

    Output:

    ```
          Name  Age      City
    0   Alice   25  New York
    1     Bob   30    London
    ```

*   **`tail()`:**  Returns the last *n* rows (default is 5).

    ```python
    print(df.tail(2)) # Output: Last 2 rows
    ```

    Output:

    ```
          Name  Age   City
    2  Charlie   27  Paris
    3    David   22  Tokyo
    ```

*   **`info()`:**  Provides a summary of the DataFrame, including data types and non-null counts.

    ```python
    print(df.info())
    ```

    Output:

    ```
    <class 'pandas.core.frame.DataFrame'>
    RangeIndex: 4 entries, 0 to 3
    Data columns (total 3 columns):
     #   Column  Non-Null Count  Dtype
    ---  ------  --------------  -----
     0   Name    4 non-null      object
     1   Age     4 non-null      int64
     2   City    4 non-null      object
    dtypes: int64(1), object(2)
    memory usage: 224.0+ bytes
    None
    ```

*   **`describe()`:**  Generates descriptive statistics (mean, std, min, max, etc.) for numerical columns.

    ```python
    print(df.describe())
    ```

    Output:

    ```
            Age
    count   4.000000
    mean   26.000000
    std     3.366502
    min    22.000000
    25%    24.250000
    50%    26.000000
    75%    27.750000
    max    30.000000
    ```

## Reading and Writing Data

Pandas can read data from various file formats:

*   **CSV:** Comma-Separated Values

    ```python
    df = pd.read_csv('data.csv')  # Reads data from a CSV file
    df.to_csv('output.csv', index=False)  # Writes data to a CSV file (without the index)
    ```

*   **Excel:**

    ```python
    df = pd.read_excel('data.xlsx')  # Reads data from an Excel file
    df.to_excel('output.xlsx', index=False)  # Writes data to an Excel file (without the index)
    ```

*   **SQL Databases:**

    ```python
    import sqlite3

    conn = sqlite3.connect('database.db')  # Connect to a SQLite database
    df = pd.read_sql_query("SELECT * FROM customers", conn) # Reads data from SQL query
    conn.close()
    ```

## Data Manipulation

Pandas provides a wide range of functions for manipulating data:

*   **Adding a new column:**

    ```python
    df['Salary'] = [50000, 60000, 55000, 48000]  # Adds a new 'Salary' column
    print(df)
    ```
    Output:
    ```
          Name  Age      City  Salary
    0    Alice   25  New York   50000
    1      Bob   30    London   60000
    2  Charlie   27     Paris   55000
    3    David   22     Tokyo   48000
    ```

*   **Dropping a column:**

    ```python
    df = df.drop('Salary', axis=1)  # Drops the 'Salary' column (axis=1 refers to columns)
    print(df)
    ```

    Output:

    ```
          Name  Age      City
    0    Alice   25  New York
    1      Bob   30    London
    2  Charlie   27     Paris
    3    David   22     Tokyo
    ```

*   **Filtering data:**

    ```python
    filtered_df = df[df['Age'] > 25]  # Filters rows where Age is greater than 25
    print(filtered_df)
    ```

    Output:

    ```
        Name  Age    City
    1     Bob   30  London
    2  Charlie   27   Paris
    ```

*   **Sorting data:**

    ```python
    sorted_df = df.sort_values(by='Age', ascending=False)  # Sorts by 'Age' in descending order
    print(sorted_df)
    ```

    Output:

    ```
          Name  Age      City
    1      Bob   30    London
    2  Charlie   27     Paris
    0    Alice   25  New York
    3    David   22     Tokyo
    ```

*   **Grouping data (using `groupby()`):**

    ```python
    grouped_data = df.groupby('City')['Age'].mean()  # Groups by 'City' and calculates the mean 'Age'
    print(grouped_data)
    ```

    Output:

    ```
    City
    London      30.0
    New York    25.0
    Paris       27.0
    Tokyo       22.0
    Name: Age, dtype: float64
    ```

*   **Handling missing values:**

    Pandas offers several functions for handling missing data (represented as `NaN`):

    *   **`fillna()`:**  Replaces missing values with a specified value.

        ```python
        import numpy as np

        data = {'A': [1, 2, np.nan, 4], 'B': [5, np.nan, 7, 8]}
        df = pd.DataFrame(data)
        print(df)

        df_filled = df.fillna(0)  # Fills NaN with 0
        print(df_filled)
        ```

        Output:

        ```
             A    B
        0  1.0  5.0
        1  2.0  NaN
        2  NaN  7.0
        3  4.0  8.0
             A    B
        0  1.0  5.0
        1  2.0  0.0
        2  0.0  7.0
        3  4.0  8.0
        ```

    *   **`dropna()`:**  Removes rows or columns containing missing values.

        ```python
        df_dropped = df.dropna()  # Drops rows with NaN
        print(df_dropped)
        ```
        Output:

        ```
             A    B
        0  1.0  5.0
        3  4.0  8.0
        ```
