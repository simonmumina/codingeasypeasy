# Mastering `borderBottomLeftRadius`: Rounded Corners in HTML with JavaScript

In the ever-evolving world of web development, aesthetics play a crucial role in user experience. Rounded corners are a common design element that can add a touch of elegance and modernity to your websites. While CSS provides powerful tools for styling elements, sometimes you need the flexibility of JavaScript to dynamically control these styles based on user interaction, data, or other application logic. This blog post dives deep into the `borderBottomLeftRadius` property of the HTML DOM Style object, empowering you to manipulate the bottom-left corner radius of elements programmatically.

## What is `borderBottomLeftRadius`?

The `borderBottomLeftRadius` property is part of the CSSStyleDeclaration object that represents the style of an HTML element. It allows you to get or set the radius of the bottom-left corner of an element's border. This property accepts values in CSS units such as pixels (`px`), percentages (`%`), ems (`em`), rems (`rem`), and others supported by CSS.

## Why Use `borderBottomLeftRadius` with JavaScript?

While you can easily set the `border-bottom-left-radius` property using CSS, manipulating it with JavaScript offers several advantages:

- **Dynamic Styling:** Modify the corner radius based on user interactions, such as hovering, clicking, or scrolling.
- **Conditional Styling:** Apply different corner radii based on conditions, such as screen size, device type, or user preferences.
- **Data-Driven Styling:** Change the radius based on data received from an API or database.
- **Animations and Transitions:** Create smooth animations and transitions by gradually changing the `borderBottomLeftRadius` value.
- **Complex UIs:** For more intricate UI designs, programmatic manipulation can be the best approach.

## Syntax

The syntax for accessing and setting the `borderBottomLeftRadius` property is straightforward:

**Getting the value:**

```plaintext
const element = document.getElementById('myElement')
const radius = element.style.borderBottomLeftRadius
console.log(radius) // Output: e.g., "10px", "20%" or an empty string if not set
```

**Setting the value:**

```plaintext
const element = document.getElementById('myElement')
element.style.borderBottomLeftRadius = '25px'
```

## Examples and Use Cases

Let's explore some practical examples of how to use `borderBottomLeftRadius` in JavaScript.

### Example 1: Setting a Fixed Radius

This is the simplest example, demonstrating how to set a fixed radius for the bottom-left corner:

```plaintext
<div id="myDiv" style="width: 200px; height: 100px; background-color: lightblue; border: 1px solid black;">
  This is a div element.
</div>

<script>
  const divElement = document.getElementById("myDiv");
  divElement.style.borderBottomLeftRadius = "15px";
</script>
```

In this example, the bottom-left corner of the `div` element will have a radius of 15 pixels.

### Example 2: Dynamic Radius Based on User Input

This example shows how to dynamically change the radius based on user input from a slider.

```plaintext
<label for="radiusSlider">Bottom-Left Radius:</label>
<input type="range" id="radiusSlider" min="0" max="50" value="0">
<div id="dynamicDiv" style="width: 200px; height: 100px; background-color: lightcoral; border: 1px solid black;">
  This div's corner changes.
</div>

<script>
  const slider = document.getElementById("radiusSlider");
  const dynamicDiv = document.getElementById("dynamicDiv");

  slider.addEventListener("input", function() {
    dynamicDiv.style.borderBottomLeftRadius = slider.value + "px";
  });
</script>
```

As the user moves the slider, the `borderBottomLeftRadius` of the `dynamicDiv` element will change in real-time.

### Example 3: Conditional Radius Based on Screen Size

This example demonstrates how to adjust the radius based on the screen size:

```plaintext
<div id="responsiveDiv" style="width: 200px; height: 100px; background-color: lightgreen; border: 1px solid black;">
  Responsive Radius
</div>

<script>
  const responsiveDiv = document.getElementById("responsiveDiv");

  function updateRadius() {
    if (window.innerWidth < 768) {
      responsiveDiv.style.borderBottomLeftRadius = "20px";
    } else {
      responsiveDiv.style.borderBottomLeftRadius = "0";
    }
  }

  // Call the function on page load and window resize
  updateRadius();
  window.addEventListener("resize", updateRadius);
</script>
```

This code sets a radius of 20px for the bottom-left corner of the `responsiveDiv` when the screen width is less than 768px (typical for mobile devices). Otherwise, it sets the radius to 0, creating a square corner.

### Example 4: Animating the Radius

This example illustrates how to create a simple animation by changing the radius over time using `setInterval`:

```plaintext
<div id="animatedDiv" style="width: 200px; height: 100px; background-color: lightsalmon; border: 1px solid black;">
  Animated Radius
</div>

<script>
  const animatedDiv = document.getElementById("animatedDiv");
  let radius = 0;
  let direction = 1; // 1 for increasing, -1 for decreasing

  setInterval(function() {
    radius += direction;

    if (radius > 30) {
      direction = -1; // Reverse direction when reaching the maximum radius
    } else if (radius < 0) {
      direction = 1; // Reverse direction when reaching the minimum radius
    }

    animatedDiv.style.borderBottomLeftRadius = radius + "px";
  }, 50); // Change the radius every 50 milliseconds
</script>
```

This code creates a continuously changing radius effect, making the bottom-left corner of the `animatedDiv` appear to pulse.

## Best Practices

- **Use Meaningful Units:** Choose appropriate units (pixels, percentages, ems, etc.) based on the desired effect and the context of your design.
- **Consider Responsiveness:** Design your corner radii with responsiveness in mind. Use percentages or media queries to adjust the radii for different screen sizes.
- **Maintain Consistency:** Ensure consistency in your use of rounded corners throughout your website for a cohesive visual experience.
- **Performance:** Avoid excessive or unnecessary manipulations of the `borderBottomLeftRadius` property, especially in performance-critical applications.

## Alternatives and Related Properties

- **CSS `border-bottom-left-radius`:** The CSS equivalent of the `borderBottomLeftRadius` property. Use this for static styling.
- **`borderTopLeftRadius`:** Controls the top-left corner radius.
- **`borderTopRightRadius`:** Controls the top-right corner radius.
- **`borderBottomRightRadius`:** Controls the bottom-right corner radius.
- **`borderRadius`:** A shorthand property for setting all four corner radii at once.
