# Chakra UI Stat Component: Displaying Key Metrics with Style and Ease

In modern web applications, presenting data effectively is crucial. Users need to quickly grasp key performance indicators (KPIs), statistics, and other vital metrics. Chakra UI provides a powerful and versatile component specifically designed for this purpose: the `Stat` component. This blog post will dive deep into the Chakra UI `Stat` component, exploring its various features, customization options, and best practices for effectively presenting data to your users.

## What is the Chakra UI Stat Component?

The `Stat` component in Chakra UI is a dedicated component designed for displaying numerical data, typically representing metrics or statistics. It provides a structured and aesthetically pleasing way to present this information, making it easy for users to understand at a glance. The `Stat` component is composed of several sub-components that work together to create a cohesive and informative display:

- **`StatGroup`:** Used to group multiple `Stat` components together, often for displaying related metrics side-by-side.
- **`StatLabel`:** Provides a descriptive label for the statistic being displayed. This is crucial for context and understanding.
- **`StatNumber`:** Displays the actual numerical value of the statistic.
- **`StatHelpText`:** Offers additional context or information about the statistic. This can be used to clarify the meaning of the number or provide a source for the data.
- **`StatArrow`:** Visually indicates whether the statistic is trending upwards or downwards.
- **`StatPercent`:** Displays the percentage change associated with the statistic, often used in conjunction with `StatArrow`.
- **`StatIcon`:** Adds a visual element related to the statistic.

## Basic Usage: Displaying a Simple Statistic

Let's start with a basic example to illustrate the core functionality of the `Stat` component. We'll display the number of users on a website:

```plaintext
import { Stat, StatLabel, StatNumber, StatHelpText, Box } from '@chakra-ui/react';

function BasicStatExample() {
  return (
    <Box p={4} maxWidth="300px" borderWidth="1px" borderRadius="md">
      <Stat>
        <StatLabel>Total Users</StatLabel>
        <StatNumber>1,456,234</StatNumber>
        <StatHelpText>Updated today</StatHelpText>
      </Stat>
    </Box>
  );
}

export default BasicStatExample;
```

In this example:

- We import the necessary components from `@chakra-ui/react`.
- We use `StatLabel` to label the statistic as "Total Users."
- `StatNumber` displays the actual number (1,456,234).
- `StatHelpText` provides context by indicating that the data was updated today.
- We wrap everything in a `Box` for visual styling (padding, border, and border radius).

## Grouping Stats with `StatGroup`

Often, you'll want to display multiple related statistics together. The `StatGroup` component makes this easy:

```plaintext
import { Stat, StatLabel, StatNumber, StatHelpText, StatArrow, StatGroup, Box } from '@chakra-ui/react';

function StatGroupExample() {
  return (
    <Box p={4} borderWidth="1px" borderRadius="md">
      <StatGroup>
        <Stat>
          <StatLabel>Subscribers</StatLabel>
          <StatNumber>71,897</StatNumber>
          <StatHelpText>
            <StatArrow type="increase" />
            23.36% increase
          </StatHelpText>
        </Stat>

        <Stat>
          <StatLabel>Avg. Open Rate</StatLabel>
          <StatNumber>55%</StatNumber>
          <StatHelpText>
            <StatArrow type="decrease" />
            1.74% decrease
          </StatHelpText>
        </Stat>
      </StatGroup>
    </Box>
  );
}

export default StatGroupExample;
```

Here, we're displaying two statistics: "Subscribers" and "Avg. Open Rate." We use `StatArrow` to indicate whether each statistic is increasing or decreasing, along with the percentage change. `StatGroup` automatically handles the layout to ensure that the stats are displayed neatly side-by-side (on larger screens). On smaller screens, they will stack vertically.

## Adding Visual Cues with `StatArrow` and `StatPercent`

The `StatArrow` and `StatPercent` components are excellent for providing visual cues about the trend of a statistic. `StatArrow` displays an arrow indicating whether the value is increasing or decreasing. `StatPercent` displays the percentage change. Let's see them in action:

```plaintext
import { Stat, StatLabel, StatNumber, StatHelpText, StatArrow, StatPercent, Box } from '@chakra-ui/react';

function ArrowPercentExample() {
  return (
    <Box p={4} maxWidth="300px" borderWidth="1px" borderRadius="md">
      <Stat>
        <StatLabel>Total Sales</StatLabel>
        <StatNumber>$1,234,567</StatNumber>
        <StatHelpText>
          <StatArrow type="increase" />
          <StatPercent>12.5%</StatPercent> increase from last month
        </StatHelpText>
      </Stat>
    </Box>
  );
}

export default ArrowPercentExample;
```

In this example, `StatArrow` displays an upward arrow, and `StatPercent` shows the 12.5% increase. The `type` prop of `StatArrow` determines the direction of the arrow. It can be either `"increase"` or `"decrease"`.

## Adding Icons with `StatIcon`

To further enhance the visual appeal and context of your statistics, you can use the `StatIcon` component. This allows you to include an icon alongside the statistic, which can help to quickly convey the meaning of the data. You'll typically use an icon library like `@chakra-ui/icons` or `react-icons`.

```plaintext
import { Stat, StatLabel, StatNumber, StatHelpText, StatIcon, Box } from '@chakra-ui/react';
import { FaUsers } from 'react-icons/fa'; // Example using react-icons

function IconExample() {
  return (
    <Box p={4} maxWidth="300px" borderWidth="1px" borderRadius="md">
      <Stat>
        <StatLabel>Website Visitors</StatLabel>
        <StatNumber>2,345,678</StatNumber>
        <StatHelpText>
          <StatIcon as={FaUsers} mr="1" />
          From last month
        </StatHelpText>
      </Stat>
    </Box>
  );
}

export default IconExample;
```

In this example:

- We import the `FaUsers` icon from the `react-icons` library.
- We use the `as` prop of `StatIcon` to specify which icon to render.
- `mr="1"` adds a small margin to the right of the icon for spacing.

## Customization and Styling

The Chakra UI `Stat` component is highly customizable. You can easily adjust the appearance to match your application's design. Chakra UI's styling props like `color`, `fontSize`, `fontWeight`, `margin`, and `padding` can be applied directly to the `Stat`, `StatLabel`, `StatNumber`, and `StatHelpText` components.

```plaintext
import { Stat, StatLabel, StatNumber, StatHelpText, Box } from '@chakra-ui/react';

function CustomStyleExample() {
  return (
    <Box p={4} maxWidth="300px" borderWidth="1px" borderRadius="md" bg="gray.50">
      <Stat>
        <StatLabel fontSize="sm" color="gray.600">
          Revenue
        </StatLabel>
        <StatNumber fontSize="2xl" fontWeight="bold" color="teal.500">
          $5,432,100
        </StatNumber>
        <StatHelpText fontSize="xs" color="gray.500">
          Year to date
        </StatHelpText>
      </Stat>
    </Box>
  );
}

export default CustomStyleExample;
```

In this example, we've customized the `fontSize` and `color` of each sub-component to create a specific visual style. We've also added a background color to the surrounding `Box`.

## Accessibility Considerations

Chakra UI prioritizes accessibility, and the `Stat` component is no exception. Ensure that your `StatLabel` provides clear and descriptive information about the statistic being displayed. This is essential for users who rely on screen readers. Additionally, consider using ARIA attributes to further enhance accessibility, especially if your statistic is dynamically updated.

## Best Practices for Using the Stat Component

- **Use descriptive labels:** `StatLabel` should clearly explain what the statistic represents.
- **Provide context:** Use `StatHelpText` to offer additional information or a source for the data.
- **Choose appropriate visuals:** Use `StatArrow` and `StatPercent` to highlight trends and changes.
- **Maintain consistency:** Use a consistent style for your `Stat` components throughout your application.
- **Consider accessibility:** Ensure that your statistics are accessible to all users.
- **Limit the amount of information:** Don't overload the Stat component. Only display the most important information. If you need to display more information, consider using a more complex component.

## Conclusion

The Chakra UI `Stat` component provides a powerful and flexible way to display key metrics in your React applications. By leveraging its various sub-components and customization options, you can create visually appealing and informative displays that help users quickly grasp important data. Remember to prioritize accessibility and use best practices to ensure that your statistics are clear, concise, and understandable for all users. With the `Stat` component, you can elevate the way you present data and create a more engaging and informative user experience.
